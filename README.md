# mirza-stream-rec
UnityでMiRZAマイクから音声をストリーミングで取得する

## ビルド方法
### はじめに
ビルドをするには、PlatformをAndroidに変更してください

### エラー
ビルドする際に写真のようなエラーが出た場合は以下を実施して再度ビルドをしてください。
1. File > BuildSettings > PlayerSettings > Project Validation に移動
2. Fix Allをクリック
